# Installation

Run this to install:
 
```sh
$ git clone https://github.com/Miguel-Galdin0/AttackOnFtp.git
```

# Usage
 
```sh
$ python3 attackonftp.py (IP) -u (user) -w (wordlist) [opitional commands]
```
